mout

Modular JavaScript Utilities

mout is a collection of modular JavaScript utilities that can be used in the browser as AMD modules or on node.js (without any overhead). It provides many helper methods similar to those found on other languages standard libraries (ie. Python, Ruby, PHP)

The biggest difference between other existing solutions is that you can load only the modules/functions that you need, no extra overhead or complex build systems required. All batteries included.

Download

mout is available as a node.js module

npm install mout

or as AMD modules to be used on browsers.

bower install mout

you can also download the AMD source files directly from github as zip or tar.

or clone the git repository directly:

git clone https://github.com/mout/mout.git
